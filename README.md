## About Application

Registrasi website dengan verifikasi email dengan mailtrap :

<h4>referensi</h4>

- [Laravel 10](https://laravel.com/docs/10.x) .
- MYSQL.
- PHP 8.1+
- [Bootsrap 5](https://getbootstrap.com/docs/5.3/getting-started/introduction/).
- [Mail Trap - Verification Email](https://mailtrap.io/).
- [Laragon 6.0](https://laragon.org/download/index.html).


## Setting

setting aplikasi : 

php artisan migrate
___________________________


setting environtment :

mailtrap: 

MAIL_MAILER=smtp<br>
MAIL_HOST=sandbox.smtp.mailtrap.io<br>
MAIL_PORT=2525<br>
MAIL_USERNAME=username di mailtrap.io<br>
MAIL_PASSWORD=password di mailtrap.io<br>
MAIL_ENCRYPTION=null<br>
MAIL_FROM_ADDRESS="namaemailkamu@gmail.com"<br>
MAIL_FROM_NAME="username"
